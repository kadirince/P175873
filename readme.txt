- to build the project please open the folder 'bin'
- to build the tests please open the folder 'test'
- to read documentation please open index.html in doc/html 

--------------------------
Tools / Frameworks:
--------------------------
1.) Qt: sudo apt-get install qt5-default
2.) Qt-Quick: sudo apt-get install qtdeclarative5-dev
3.) cmake: sudo apt-get install cmake
4.) astyle: sudo apt-get install astyle
5.) cppcheck: sudo apt-get install cppcheck
6.) doxygen: sudo apt-get install doxygen doxygen-doc doxygen-gui graphviz
7.) lcov: sudo apt-get install lcov
